Sys calls related to the program execution flow.

Provides API for low-level async implementation.

Functions

exit
Terminate the execution of a program. The program and all corresponding data are removed from the storage. This is similar to std::process::exit. value_destination specifies the address where all available to the program value should be transferred to. Maybe called in init method as well.
gas_available
Get the current value of the gas available for execution.

random

wait
Pause the current message handling.
wait_for
Same as wait, but delays handling for given specific amount of blocks.
wait_up_to
Same as wait, but delays handling for maximal amount of blocks that could be payed, that doesn’t exceed given duration.
wake
Resume previously paused message handling.
wake_delayed
